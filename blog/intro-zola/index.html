ホームページに Zola を導入した

created at 2025/01/23 , updated at 2025/06/17

ホームページに Jamstack の一つである Zola を導入した．

Attaching theme

  • theme には Hook を用いた．
    • config.toml に theme を設定し，content/ と content/blog にそれぞれ _index.md を置けば動く．

Deploy using shalzz/zola-deploy-action

  • Custom Domain を設定する場合は static/CNAME ファイルに使用したいドメインを書く
  • Deploy key の有効期限が切れたときは tokens にアクセスして鍵をアップデート． その後，自分のリポジトリの settings/secrets/actions からGITHUB_TOKEN をアップデート
    • Generate New Token(classic) から public_repo にチェックをつけてTokenを生成する．
